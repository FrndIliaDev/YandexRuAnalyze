# YandexRuAnalyze

- Определите, на каком протоколе работает сайт.
- Проанализируйте структуру страницы сайта.
- Внесите не менее 10 изменений на страницу с помощью инструмента разработчика и представьте скриншоты было/стало.
- Создайте прототип низкой детализации (дополнительное задание, если на семинаре дошли до задания №8).



1. Протокол работы сайта
Сайт yandex.ru работает по протоколу HTTPS (HyperText Transfer Protocol Secure). Это подтверждается:
URL в адресной строке начинается с https://
Наличие SSL/TLS сертификата (значок замка в адресной строке)
Шифрование передачи данных для безопасности пользователей

3. Анализ структуры страницы
Структура главной страницы Яндекса:
Основные блоки:

Header (Верхняя панель):
Логотип Яндекс
Поисковая строка
Кнопка "Найти"
Меню сервисов (почта, карты, маркет и др.)
Кнопка входа в аккаунт

Main Content (Основное содержимое):
Виджеты (погода, пробки, курс валют и т.д.)
Быстрые ссылки на сервисы
Новостная лента
Рекламные баннеры (в зависимости от региона)

Footer (Нижняя панель):
Ссылки на разные версии Яндекса
Информация о компании
Языковые настройки
Ссылка на Яндекс.Браузер

Технические особенности:
Используется современная семантическая верстка
Адаптивный дизайн (подстраивается под разные устройства)
Много JavaScript для динамического обновления контента
Используются React-компоненты (на основе анализа классов)
